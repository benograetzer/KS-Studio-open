KS Studio AR Viewer

Dieses Repository hostet ausschliesslich oeffentliche AR- und 3D-Modelle fuer die Anzeige im Browser. Es enthaelt keine internen Konfigurator-, Preis-, ERP-, Kunden- oder Projektdaten.

AR-Viewer oeffnen

Modelle per URL laden

GLB-Dateien werden ueber den Parameter model geladen:

ar-viewer.html?model=exports/dateiname.glb

Optional kann fuer iPhone/iPad Quick Look eine USDZ-Datei ergaenzt werden:

ar-viewer.html?model=exports/test.glb&ios=exports/test.usdz
